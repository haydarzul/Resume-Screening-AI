About This Project

Everything you need to know about the Resume Screening AI

  • Dataset: Kaggle Resume Classification Dataset
  • Model: Multinomial Naive Bayes
  • Accuracy: ~97.9%
  • File Support: PDF, DOCX, and TXT
  • Vectorizer: TF-IDF (Text Feature Extraction)
  • Purpose: Automatically classify resumes into job roles
